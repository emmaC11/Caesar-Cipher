How to use a Caesar Cipher

A caesar cipher converts plain text into in an encoded message - this is known as encrypting

The plain text is encrypted into ciphertext. The ciphertext looks like a combination of random letters

The original text cannot be identified by reading the cyphertext

The key is the number of times you want to shift the letters.

Ciphertext can only be decrypted if you know the key.
